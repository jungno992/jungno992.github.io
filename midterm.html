인라인 요소 목록

다음은 HTML에서 자주 사용되는 요소들 입니다.

  • a : 하이퍼링크 (Anchor)
  • img : 이미지 (Image)
  • span : 일반용도의 인라인 (Span)
  • strong : 중요 (Strong)
  • em : 강조 (Emphasis)

HTML이란?

HTML은 하이퍼텍스트 마크업 언어(Hypertexe Markup Language)의 약자로, 웹 페이지를 만들 때 사용되는 문서 형식입니다.

하이퍼텍스트(Hypertexe)란 하이퍼링크(Hyperlink)를 담고 있는 문서를 뜻합니다,. 하이퍼링크는 문서들이 서로 연결되어 있는 고리를 말합니다.

CSS란?

CSS는 Cascading Stylesheet의 약자이며 HTML 같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다. HTML 문서의 head요소 안에style요소를 넣고 CSS 코드를 적으면 해당 문서를 다양하게 꾸밀 수 있습니다.

CSS에 대한 더 자세한 설명은 CSS 문서를 참고하세요.

이미지 넣기

이미지를 넣으려면 img 요소를 사용합니다.

src 속성에는 이미지의 결로(URL)를 적습니다.

이미지 요소에는 항상alt속성을 사용하여 대체 텍스트를 명시해주어야 합니다. "alt"는 "Alternative text"의 약자로, 이미지를 표시할 수 없을 때 이미지를 대신하여 나타낼 텍스트를 뜻합니다.
